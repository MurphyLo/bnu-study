# bnu-study
***library reverie***

### requirements

- `pyexecjs`
- `requests`
- `bs4`
- `des.js`

### how to

1. Install required packages.
2. Modify `main.py`
    - fill in `u` and `p`.
    - change `start`, `end` to your desired time range for reverie.
3. Make sure `main.py`, `des.js` are in the same directory and run `main.py` **before 19:30**.
